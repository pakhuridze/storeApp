# Simple E-Commerce API

## Overview

This is a simple E-Commerce API.
Using Python, Django and MPTT
for category tree structure.

### Features

- product (json)
- product can have many categories
- category (json)
- category have tree structure

## Installation

```
pip install -r requirements.txt
```
